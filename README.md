# Bryan Mayodi — Portfolio

My personal portfolio website, a fullstack developer focused on frontend development, user experience, and building scalable systems.

Live at: [bryanmayodi.vercel.app](https://bryan-mayodi.vercel.app)

## Stack

- React 19 + TypeScript
- Tailwind CSS v4
- Framer Motion
- Vite
- Formspree (contact form)
- react-icons

## Sections

- Hero — animated typing effect with role cycling
- About — bio and skills
- Projects — featured work with links
- Experience — career timeline
- Contact — working contact form via Formspree

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
